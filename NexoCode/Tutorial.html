Guía para comenzar con NexoCode

Sigue estos pasos para crear tu primer repositorio y subir tu proyecto.

1. Abre tu proyecto

Dirígete a la carpeta donde tienes tu proyecto en tu ordenador.

2. Abre una terminal

Puedes usar PowerShell, CMD o la terminal integrada de tu editor.

3. Entra en la carpeta del proyecto

Usa el siguiente comando:

cd ruta/de/tu/proyecto

4. Inicializa el repositorio con Nexo

Ejecuta:

nexo init

5. Agrega tus archivos

Para agregar todos los archivos del proyecto:

nexo add .

6. Crea tu primer commit

nexo commit -m "Primer commit"

7. Sube tu proyecto a NexoCode

Conecta tu repositorio local con el remoto:

nexo remote add origin <URL_DEL_REPOSITORIO>

Luego súbelo:

nexo push -u origin main
